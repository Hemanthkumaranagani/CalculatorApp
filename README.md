# CalculatorApp

## Description

A Java console-based calculator that performs basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Exception handling for invalid inputs
* Division-by-zero protection

## Technologies Used

* Java
* IntelliJ IDEA

## How to Run

1. Compile the program.
2. Run Calculator.java.
3. Enter two numbers and an operator when prompted.
