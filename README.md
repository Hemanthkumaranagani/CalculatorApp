# CalculatorApp

## Description

CalculatorApp is a simple Java console-based calculator that performs basic arithmetic operations. The application allows users to enter two numbers and select an operator to calculate the result.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* User input through the Scanner class
* Switch-case statements for operation selection
* Exception handling for invalid inputs
* Division-by-zero protection
* Supports whole numbers, negative numbers, and decimal values

## Technologies Used

* Java
* IntelliJ IDEA
* Git & GitHub

## How to Run

1. Open the project in IntelliJ IDEA.
2. Compile and run `Calculator.java`.
3. Enter the first number.
4. Enter an arithmetic operator (+, -, *, /).
5. Enter the second number.
6. View the calculated result.

## Testing

The application has been tested with:

* Positive numbers
* Negative numbers
* Decimal values
* Division by zero
* Invalid operators
* Non-numeric input

## Learning Outcomes

This project demonstrates:

* Java programming fundamentals
* User input handling with Scanner
* Decision-making using switch-case statements
